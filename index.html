📌 About the Bot

The AI Moderation Bot is designed to make Discord servers safer and smarter. It uses ai.js, prompt.js, and deploy-commands.js to integrate Hugging Face AI (DeepSeek / Mistral) for moderation logic.

⚡ Key Features

  • 🚨 AI-powered moderation system
  • 💬 Detects toxicity, spam, and harmful messages
  • 🤖 Smart AI replies using Hugging Face models
  • ⚙️ Hardcoded config via prompt.js
  • 🛠️ Easy to extend with new commands

📂 Source Code

Original repository: 👉 Moderation-Bot-With-AI-Logic (GitHub)

🚀 How to Run

  1. Clone the repo: git clone https://github.com/Jackpk52/Moderation-Bot-With-AI-Logic
  2. Install dependencies: npm install
  3. Edit prompt.js with your bot token + Hugging Face key
  4. Run the bot: node index.js
